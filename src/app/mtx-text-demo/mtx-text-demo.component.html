Mtx Text (MtxTextModule) Mtx-Text is a way of arranging type to make text legible, readable, and appealing when displayed. Mtx-Text is based on the guidelines from the Material Design spec and is arranged into multiple tags. Each tag has a font-size, line-height and font-weight. The available levels are: Attribute Description mtx-display Large, one-off header or a hero header). mtx-headline Useful as section headings. mtx-title Titles can be used in toolbars, cards etc. mtx-subtitle Can be used instead of Titles for lower emphasis. mtx-label Bold text generally used in buttons. mtx-paragraph Regular paragraph text. mtx-caption Smaller body and hint text. mtx-overline Smallest text generally used as overlines.
