Mtx App Bar MtxAppBarModule

Mtx App Bar is a toolbar that displays information and actions relating to the current UI. It is based on the guidelines from the Material Design specifications and is arranged into multiple tags. Each tag has a different size and are described as under:

Attribute Description {{entry.attribute}} {{entry.description}} Basic Usage <{{entry.attribute}}>Title</{{entry.attribute}}> {{entry.title}} Title Navigation Mtx App Bar supports navigation and is managed by the nav property. It supports the following values and are described as under: Value Description {{entry.attribute}} {{entry.description}} App Bar nav property <mtx-app-bar class="mtx-primary" app-bar-title="App Bar with menu" nav="menu"></mtx-app-bar> Title Mtx App Bar can have a title and is managed by the app-bar-title or appBarTitle property which supports a string value. For custom titles, mtx-app-bar-title or h1 mtx-app-bar-title sub-component can be used. App Bar title sub component <mtx-app-bar class="mtx-primary">     <h1 mtx-app-bar-title>My New Page</h1> </mtx-app-bar>

 Action Buttons Mtx App Bar can have action buttons on the right and can be defined inside mtx-app-bar carrying a action selector. It supports all HTML elements especially MatButtons App Bar with actions <mtx-app-bar class="mtx-primary" nav="close">     <button mat-button>SAVE</button> </mtx-app-bar> Overflow Mtx App Bar can have an overflow menu is managed by the overflow property which supports a MatMenu Examples Dense App Bar with overflow <mtx-dense-app-bar class="mtx-primary" app-bar-title="Title" nav="back" overflow="menu"></mtx-dense-app-bar> <mat-menu #menu></mat-menu> Standard App Bar with buttons <mtx-app-bar class="mtx-primary" app-bar-title="Title" nav="menu">     <button mat-icon-button>         <mat-icon>search</mat-icon>     </button>     <button mat-icon-button>         <mat-icon>share</mat-icon>     </button> </mtx-app-bar> Prominent App Bar <mtx-app-bar class="mtx-primary" nav="menu">     <button mat-icon-button>         <mat-icon>search</mat-icon>     </button>     <button mat-icon-button>         <mat-icon>share</mat-icon>     </button> </mtx-app-bar> <mtx-toolbar prominent>Prominent title</mtx-toolbar> Prominent title ...
