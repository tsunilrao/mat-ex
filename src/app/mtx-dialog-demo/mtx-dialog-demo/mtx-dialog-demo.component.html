<mtx-dialog> Displays dialogs with Material Design styling and animations. Dialog Overview This is the Dialog text
Attributes There are several dialog variants, each applied as an attribute: Attribute Description mtx-dialog Rectangular contained dialog without elevation mtx-raised-dialog Rectangular contained dialog with elevation mtx-stroked-dialog Rectangular outlined dialog without elevation Directives Several directives are available to make it easier to structure your dialog: Directive Description mtx-dialog-title Container for title bar at the top of the dialog. mtx-dialog-content Container for all the content of the dialog. mtx-dialog-actions Container for action buttons at the bottom of the dialog. Button alignment can be controlled via the align attribute which can be set to left, right and center.
